Description

L'objectif de ce projet est un site Web Front-End de magasin en utilisant HTML/CSS et Javascript.
Le site Web présentera certains produits ou services qu'une entreprise souhaite vendre ou promouvoir.
Le site Web aura une page d'accueil, une page de produits et une page de contact. Le site sera réactif et accessible sur différents appareils et navigateurs.

Fonctionnalités

  • La page d'accueil présente le magasin et ses produits ou services, avec des images et du texte.
  • La page de produits affiche les détails de chaque produit ou service, tels que le nom, le prix, la description, les caractéristiques, les avis.
  • La page de contact indiquant l'adresse du magasin, son numéro de téléphone, son adresse électronique, les liens vers les médias sociaux.
  • La page decontact comporte aussi une section FAQ répondant aux questions que les clients pourraient se poser.
